# Azure Basic Workshop

Azure Basic Workshop은 애저 인프라 코어 서비스들을 직접 Hands on Lab 할 수 있는 컨텐츠입니다.
소요 시간에 따라 Basic / Advanced 두 가지 컨텐츠로 구성되어 있습니다.
Basic의 경우 약 한 시간, Advanced의 경우 약 두 시간이 소요됩니다.
